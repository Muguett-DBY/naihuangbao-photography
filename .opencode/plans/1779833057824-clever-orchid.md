# 超大升级方案：个人摄影师站 → 品牌化多页面平台

## 现状

- **架构**: 单页长滚动（20+ section），React 19 + Vite 8 + Cloudflare Pages/Workers/D1/R2
- **功能**: 作品集、预约系统、AI 聊天、评价、地图、4 语言 i18n
- **动画**: GSAP 20 种滚动特效 + Framer Motion + Three.js 3D 照片墙
- **痛点**: 页面过长、功能堆叠、难以扩展新业务线

## 目标

1. **多页面架构**: 6-7 个独立页面，React Router 路由
2. **新业务线**: 课程/教程、后期预设、工作坊、实体产品
3. **高端品牌设计**: Apple/法拉利级质感，保留奶黄包暖色调
4. **测试驱动**: 每次 push 前完整验证

### 1.3 路由配置

```tsx
// src/router.tsx
import { createBrowserRouter } from 'react-router-dom';
import { RootLayout } from './layouts/RootLayout';
import { HomePage } from './pages/HomePage';
import { GalleryPage } from './pages/GalleryPage';
import { CoursesPage } from './pages/CoursesPage';
import { ProductsPage } from './pages/ProductsPage';
import { WorkshopsPage } from './pages/WorkshopsPage';
import { ShopPage } from './pages/ShopPage';
import { BookingPage } from './pages/BookingPage';

export const router = createBrowserRouter([
  {
    path: '/',
    element: <RootLayout />,
    children: [
      { index: true, element: <HomePage /> },
      { path: 'gallery', element: <GalleryPage /> },
      { path: 'courses', element: <CoursesPage /> },
      { path: 'products', element: <ProductsPage /> },
      { path: 'workshops', element: <WorkshopsPage /> },
      { path: 'shop', element: <ShopPage /> },
      { path: 'booking', element: <BookingPage /> },
      { path: 'admin/*', element: <AdminLayout /> },
    ],
  },
]);
```

### 1.5 动画迁移策略

**保留现有特效**:
- GSAP 滚动动画 → 按页面拆分到各 Page 组件
- Three.js 3D 照片墙 → 仅 GalleryPage 加载
- 自定义光标 → RootLayout 全局保留
- 电影颗粒 → RootLayout 全局保留
- Lenis 平滑滚动 → RootLayout 全局保留

**优化点**:
- 拆分 `useGsapAnimations.ts`（671 行）为独立 effect 文件
- 每个页面只注册自己的 GSAP effects，避免全局冲突

## Phase 2: 新数据模型

### 2.1 课程/教程表

```sql
CREATE TABLE courses (
  id TEXT PRIMARY KEY,
  title TEXT NOT NULL,
  title_en TEXT,
  title_ko TEXT,
  title_ja TEXT,
  description TEXT,
  description_en TEXT,
  description_ko TEXT,
  description_ja TEXT,
  cover_image_url TEXT,
  video_url TEXT,           -- Bilibili/YouTube embed URL
  content_markdown TEXT,    -- 图文教程内容
  category TEXT,            -- 'beginner'|'advanced'|'post-processing'|'posing'
  difficulty TEXT,          -- 'beginner'|'intermediate'|'advanced'
  duration_minutes INTEGER,
  sort_order INTEGER DEFAULT 0,
  published INTEGER DEFAULT 0,
  created_at TEXT,
  updated_at TEXT
);

CREATE TABLE course_modules (
  id TEXT PRIMARY KEY,
  course_id TEXT NOT NULL,
  title TEXT NOT NULL,
  title_en TEXT,
  title_ko TEXT,
  title_ja TEXT,
  type TEXT NOT NULL,       -- 'video'|'text'|'gallery'
  content TEXT,             -- markdown or video embed code
  sort_order INTEGER DEFAULT 0,
  FOREIGN KEY (course_id) REFERENCES courses(id)
);
```

### 2.2 预设/资源包表

```sql
CREATE TABLE presets (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  name_en TEXT,
  name_ko TEXT,
  name_ja TEXT,
  description TEXT,
  description_en TEXT,
  description_ko TEXT,
  description_ja TEXT,
  category TEXT,            -- 'lightroom'|'photoshop'|'capture-one'|'luts'
  preview_images TEXT,      -- JSON array of preview image URLs
  download_url TEXT,        -- R2 object key or external link
  price_display TEXT,       -- 展示价格（如 "¥29"）
  featured INTEGER DEFAULT 0,
  download_count INTEGER DEFAULT 0,
  created_at TEXT,
  updated_at TEXT
);
```

### 2.3 工作坊/活动表

```sql
CREATE TABLE workshops (
  id TEXT PRIMARY KEY,
  title TEXT NOT NULL,
  title_en TEXT,
  title_ko TEXT,
  title_ja TEXT,
  description TEXT,
  description_en TEXT,
  description_ko TEXT,
  description_ja TEXT,
  cover_image_url TEXT,
  event_date TEXT NOT NULL,
  event_time TEXT,
  location TEXT,
  max_participants INTEGER,
  current_participants INTEGER DEFAULT 0,
  price_display TEXT,
  status TEXT DEFAULT 'upcoming',  -- 'upcoming'|'ongoing'|'completed'|'cancelled'
  registration_form_url TEXT,      -- 外部表单链接或内置表单
  created_at TEXT,
  updated_at TEXT
);
```

### 2.4 实体产品表

```sql
CREATE TABLE merchandise (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  name_en TEXT,
  name_ko TEXT,
  name_ja TEXT,
  description TEXT,
  description_en TEXT,
  description_ko TEXT,
  description_ja TEXT,
  images TEXT,              -- JSON array of image URLs
  category TEXT,            -- 'album'|'postcard'|'frame'|'print'|'other'
  price_display TEXT,
  available INTEGER DEFAULT 1,
  created_at TEXT,
  updated_at TEXT
);
```

### 2.5 报名表

```sql
CREATE TABLE workshop_registrations (
  id TEXT PRIMARY KEY,
  workshop_id TEXT NOT NULL,
  name TEXT NOT NULL,
  contact TEXT NOT NULL,    -- 手机号或微信
  participants INTEGER DEFAULT 1,
  notes TEXT,
  status TEXT DEFAULT 'pending',  -- 'pending'|'confirmed'|'cancelled'
  created_at TEXT,
  FOREIGN KEY (workshop_id) REFERENCES workshops(id)
);
```

## Phase 3: 新 API 端点

### 3.1 课程 API

| 方法 | 路径 | 用途 |
|------|------|------|
| GET | `/api/courses` | 公开课程列表（仅 published） |
| GET | `/api/courses/:id` | 课程详情 + modules |
| POST | `/api/admin/courses` | 创建课程 |
| PATCH | `/api/admin/courses/:id` | 更新课程 |
| DELETE | `/api/admin/courses/:id` | 删除课程 |
| POST | `/api/admin/courses/:id/modules` | 添加模块 |
| PATCH | `/api/admin/courses/:id/modules/:moduleId` | 更新模块 |
