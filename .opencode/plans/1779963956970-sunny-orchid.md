# 修复方案：编辑器页面遮挡问题

## 问题根因
FilmGrain 在 RootLayout 中渲染在 Outlet **之前**，导致：
1. RootLayout 渲染 → FilmGrain 先渲染（此时 data-page 未设置）
2. Outlet 渲染 → PhotoEditorPage 挂载 → useEffect 设置 data-page
3. FilmGrain 是兄弟组件，不会因 data-page 变化重新渲染

## 修复方案

### 文件 1: src/layouts/RootLayout.tsx
- 已有 `location = useLocation()`
- 用 `location.pathname !== "/editor"` 条件渲染 FilmGrain 和 CustomCursor

### 文件 2: src/components/FilmGrain.tsx
- 移除 `data-page` 检查逻辑（不再需要）

### 文件 3: src/components/CustomCursor.tsx
- 移除 `data-page` 检查逻辑（RootLayout 已处理）

### 文件 4: src/pages/PhotoEditorPage.tsx
- 移除 `useEffect` 设置 `data-page` 的逻辑
