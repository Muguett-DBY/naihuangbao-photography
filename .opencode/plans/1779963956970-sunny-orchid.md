# 第四轮超级大升级 — 9 个新功能

## Phase 1: Before/After 对比滑块 | 复杂度 S

**文件**：新建 `src/components/CompareSlider.tsx`
**使用**：PresetDetailPage.tsx、GalleryPage.tsx

- CSS `clip-path` + pointer events 实现拖拽分割线
- 左侧原图，右侧修图效果
- 触摸/鼠标拖拽交互
- 响应式设计

## Phase 2: 预约日历视图 | 复杂度 M

**文件**：新建 `src/components/BookingCalendar.tsx`
**修改**：`src/components/BookingModal.tsx`、`functions/api/availability.ts`

- 自定义日历网格组件（不用第三方库）
- 查询 D1 `booking_requests` 表获取已预约日期
- 颜色编码：可选/已满/部分
- 月导航（Swiper 或自定义）
- API 端点：GET /api/availability?month=YYYY-MM

## Phase 3: 用户中心面板 | 复杂度 L

**文件**：新建 `src/pages/DashboardPage.tsx`
**修改**：`src/router.tsx`、`src/components/shared/Header.tsx`

- 登录用户专属页面 `/dashboard`
- 显示：预约记录、购买记录、课程进度、工坊报名
- API 端点：GET /api/user/bookings、GET /api/user/purchases、GET /api/user/courses

## Phase 7: 风格问卷推荐 | 复杂度 M
