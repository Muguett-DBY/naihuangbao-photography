# 修图工具超级升级计划

## 核心问题
当前修图工具每次拖动滑块都会重新检测人脸+应用全部效果，导致严重卡顿。需要彻底重构。

## 升级方案

### 1. 实时预览架构重构
- 将人脸检测结果缓存（只检测一次）
- 滑块变化时只重新绘制 Canvas，不重新检测
- 使用 requestAnimationFrame + 节流（30fps）
- 分层渲染：底层原图 → 中间效果层 → 顶层 UI

### 2. 撤销/重做系统
- 维护状态历史栈（最多 20 步）
- 每次参数变化保存快照
- Ctrl+Z / Ctrl+Y 快捷键

### 3. Before/After 对比
- 拖拽分割线对比原图/修图效果
- 使用 clip-path 裁剪

### 4. 一键自动美颜
- 基于人脸关键点智能计算参数
- 皮肤区域自动检测并平滑
- 面部比例自动优化

### 5. 滤镜预设库（12 种）
- 自然、美颜、人像、复古
- 电影、日系、港风、胶片
- 黑白、冷调、暖调、高对比

### 6. 更多面部调整工具
- 脸型调整、眼距调整、脸长调整
- 颧骨调整、下巴调整、人中调整

### 7. 算法升级
- 简单高斯模糊 → 双边滤波（保边去噪）
- 基本网格变形 → 基于关键点的平滑变形

### 8. 导出选项
- 质量选择（80%/90%/100%）
- 格式选择（PNG/JPEG）
- 尺寸选择（原图/缩小）

### 9. 性能优化
- Canvas 绘制去抖动（16ms 节流）
- 图像数据缓存
- 懒加载 face-api.js 模型
