# 智能一键美颜升级

## 当前问题
`handleAutoEnhance` 使用硬编码固定参数，不分析实际人脸特征。

## 升级方案
根据人脸关键点 + 肤色分析 + 亮度计算，动态生成最优参数。

### 算法设计

1. **肤色亮度分析**
   - 取面部中心区域（排除眼睛/嘴巴）的平均 RGB
   - 转换为 HSL，提取亮度值 L
   - L < 40 → 高美白 (whiten: 40)
   - L 40-60 → 中美白 (whiten: 25)
   - L > 60 → 低美白 (whiten: 15)

2. **脸型比例分析**
   - 计算脸宽/脸高比 (aspect ratio)
   - ratio > 0.75 → 脸偏圆，增加瘦脸 (slim: 25)
   - ratio 0.6-0.75 → 标准脸型 (slim: 15)
   - ratio < 0.6 → 脸偏长，减少瘦脸 (slim: 8)

3. **眼部比例分析**
   - 计算眼宽/脸宽比
   - ratio < 0.2 → 小眼，增加大眼 (bigeye: 18)
   - ratio 0.2-0.3 → 标准 (bigeye: 12)
   - ratio > 0.3 → 大眼，减少 (bigeye: 5)

4. **鼻部比例分析**
   - 计算鼻宽/脸宽比
   - ratio > 0.35 → 宽鼻，增加瘦鼻 (nose: 15)
   - 否则 → (nose: 8)

5. **皮肤平滑度**
   - 取皮肤区域的亮度方差
   - 方差大 → 皮肤不均匀，增加磨皮 (smooth: 65)
   - 方差小 → 皮肤均匀，减少磨皮 (smooth: 45)
