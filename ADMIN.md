# 后台管理说明（ADMIN.md）

## 进入后台

- 访问 `https://shoot.custard.top/admin`
- 使用管理员密码登录
- 密码通过 `wrangler pages secret put ADMIN_PASSWORD` 配置

## 功能

### 上传照片

1. 点击「选择文件」选取照片（支持 JPEG、PNG、WebP，最大 10MB）
2. 填写作品标题
3. 选择风格分类
4. 填写拍摄地点
5. 勾选「客人已授权公开展示」
6. 可选：勾选「设为精选作品」
7. 点击「上传到 Cloudflare」

### 删除照片

1. 在「当前公开作品」列表中找到要删除的照片
2. 点击右侧红色垃圾桶按钮
3. 在弹出的确认框中点击「确认删除」
4. 删除后不可恢复

### 退出登录

点击「退出登录」按钮退出

## 常见问题

### 上传失败

- 检查文件格式（仅支持 JPEG、PNG、WebP）
- 检查文件大小（需小于 10MB）
- 检查网络连接
- 检查 Cloudflare R2/D1 绑定是否正常

### 删除失败

- 检查网络连接
- 刷新页面后重试
- 已上传到 R2 但 D1 记录未删除的情况不会阻塞操作（R2 会留孤儿文件）

## 技术说明

- 后台使用 HMAC-SHA256 cookie session 认证，有效期 30 天
- 所有 API 都需携带登录 cookie
- 上传文件存于 Cloudflare R2，元数据存于 D1
